
# Jai API

Using Jai API, Julia user can easily run CPU or GPU codes written in Fortran/C/C++.


# list of Jai API

Jai API is consist of serveral Julia macros.

```@index
Modules = [AccelInterfaces]
Order   = [:constant, :type, :function, :macro]
```
